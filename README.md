# README

To run this app

#### Remove the Gemfile.lock
$ `rm Gemfile.lock`
#### Build the docker  
$ `docker-compose build`
#### Start the docker 
$ `docker-compose up`
* Ruby version
#### Create the database
$ `rails db:create`
#### Migrate the database
$ `rails db:migrate`

#### Open in the browser
http://localhost:3030/

#### Close down the docker
$ `docker-compose down`
